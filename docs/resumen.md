# IRIS Bot — Resumen Ejecutivo

## Objetivo

Asistente virtual disponible 24/7 que permite a los empleados consultar bases de datos, políticas y procedimientos mediante conversaciones en lenguaje natural.

## Qué hace hoy

| Capacidad | Descripción |
|-----------|-------------|
| Consultas a BD | Preguntas en español que se traducen a SQL automáticamente |
| Base de Conocimiento | Acceso semántico a políticas, procedimientos y manuales |
| Control de Acceso | Autenticación de empleados con restricciones por rol |
| Validación de Consultas | Bloqueo de operaciones no autorizadas |
| Interfaz Telegram | Acceso sin instalación adicional |
| API REST | Integración en apps web, móviles y portales corporativos |
| Encriptación AES | Compatible con plataformas C# legacy |
| Respuestas Contextuales | Respuestas personalizadas con tono amigable |

## Próximos pasos

- Configuración del ambiente de producción (servidor, seguridad, respaldos)
- Pruebas de aceptación (UAT) con grupo piloto
- Monitoreo y alertas de rendimiento
- Capacitación y comunicación de lanzamiento a empleados

---

**Valor de Negocio:** Reduce el tiempo de búsqueda de información, elimina dependencias técnicas y democratiza el acceso a datos corporativos.
