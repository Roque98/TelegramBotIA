# Resultados de Pruebas — Tools de Alertas

Carpeta para registrar los resultados de cada pregunta de prueba
definida en `../tools-alertas-preguntas.md`.

## Archivos

| Archivo | Sección |
|---|---|
| `01-resumen-general.md` | Preguntas generales: ¿hay alertas?, ¿cuántos equipos? |
| `02-filtrar-por-equipo-ip.md` | Filtrar alertas por nombre de equipo o IP |
| `03-analisis-profundo.md` | Análisis con LLM: diagnóstico, causa raíz, acciones |
| `04-historial-tickets.md` | Tickets históricos y acciones correctivas previas |
| `05-escalamiento-contactos.md` | Matriz de escalamiento y datos de contacto |
| `06-inventario.md` | Inventario: área, OS, ambiente, tipo, impacto |
| `07-detalle-completo.md` | Detalle completo de un equipo (multi-tool) |
| `08-formato-y-casos-borde.md` | Refinamiento de formato y casos sin datos |

## Cómo usar

1. Ejecuta la pregunta en el bot
2. Pega la respuesta en el bloque de código del archivo correspondiente
3. Anota en **Observaciones** qué mejorar (formato, datos faltantes, respuesta incorrecta, etc.)
4. Al final de cada archivo hay una sección **Notas generales** para resumir los ajustes que se hicieron
